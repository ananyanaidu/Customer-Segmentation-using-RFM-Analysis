
# 📊 Customer Segmentation using RFM Analysis


## 📝 Project Overview
 The goal of this project is to analyze transactional customer data and segment the customer base using **RFM (Recency, Frequency, Monetary)** analysis. 

By categorizing customers based on their purchasing behavior, businesses can tailor their marketing strategies, improve customer retention, and maximize revenue.

## 🎯 Objectives
*   **🧹 Data Preparation:** Clean and prepare transactional customer data for analysis.
*   **🧮 RFM Calculation:** Calculate Recency, Frequency, and Monetary metrics for each customer.
*   **👥 Segmentation:** Group customers into actionable segments (e.g., Champions, Loyal, At Risk, New Customers).
*   **📈 Analysis:** Analyze the behavioral patterns and demographics of each segment.
*   **📊 Visualization:** Build a comprehensive dashboard to visualize the segments and business insights.

## 🛠️ Tools Used
*   **Power BI:** Used for data visualization, building the interactive dashboard, and deriving insights.

## 💡 Key Insights & Findings

Based on the interactive dashboard analysis, several critical business insights were discovered:

## 🧠 What is RFM Analysis?

RFM analysis is a proven marketing model for behavior-based customer segmentation. It groups customers based on their transaction history – how recently, how often, and how much did they buy.

> [NOTE]
> **Why use RFM?** Instead of analyzing the entire customer base as a monolith, RFM allows marketers to target specific clusters of customers with communications that are much more relevant for their particular behavior.

### The Three Pillars of RFM:
1.  ⏱️ **Recency (R):** Days since last purchase. *Lower is better.*
2.  🔄 **Frequency (F):** Total number of purchases. *Higher is better.*
3.  💰 **Monetary Value (M):** Total money spent. *Higher is better.*
